# 💰 Crypto Market ETL + Live Dashboard

Python project για **ανάκτηση, ανάλυση και live visualization** των τιμών κρυπτονομισμάτων BTC, ETH και SOL.  
Μπορείς να τρέξεις το project απευθείας με Python και να δεις live dashboard στον browser.

---

## 🚀 Features

- Fetch δεδομένων από το **CryptoCompare API**  
- Καθαρισμός & μετατροπή δεδομένων σε **Pandas DataFrame**  
- Υπολογισμός **KPIs** (μέση τιμή, max/min, αλλαγή 24h, κεφαλαιοποίηση, όγκος)  
- Στατικό **bar chart** με τιμές και χρωματισμό  
- **Live dashboard** με ανανέωση κάθε 5 δευτερόλεπτα  
- Επαγγελματικό layout με **Streamlit**  
- Εύκολο setup και τρέξιμο με Python
